# Aula 00 — Introdução ao SQL no Dia a Dia de Relatórios

## Por que SQL é essencial no trabalho?
Em ambientes corporativos, dados não estão em planilhas prontas.
Eles vivem em bancos de dados que alimentam sistemas como CRM, ERPs e plataformas internas.

SQL é a ferramenta que permite:
- Buscar dados específicos
- Filtrar períodos, equipes e status
- Consolidar informações para relatórios e dashboards

---

## Exemplo prático
Perguntas comuns no dia a dia:
- Quantos registros foram criados hoje?
- Qual colaborador teve maior volume de atividades na semana?
- Como está a evolução de produtividade ao longo do mês?

Essas perguntas **não são respondidas manualmente**.
Elas exigem consultas SQL.

---

## SQL no fluxo real
1. O colaborador registra uma atividade no sistema
2. O dado é armazenado no banco
3. SQL extrai e organiza essas informações
4. Power BI consome os dados tratados
5. A gestão analisa e decide

---

## Conclusão
Antes de escrever SQL, é fundamental entender:
- Quem usa a informação
- Para que ela será usada
- Qual decisão será tomada a partir dela

Esse entendimento guiará todas as consultas dos próximos dias.
